1. CSS(cascading Style Sheets) es un lenguaje que sirve para describir la presentacion, el "estilo" de los documentos HTML

2. el CSS sirve para hacer lenguajes informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc.

3. Una regla CSS se compone de un selector y una o más declaraciones que modifican la apariencia de los elementos HTML.
La regla se utiliza para aplicar un estilo específico a los elementos seleccionados

4. Una regla de CSS está formada por una parte llamada “selector” y otra parte llamada “declaración” (puede haber una o varias dentro del mismo selector).
La declaración indica “qué hay que hacer” y el selector indica “a quién hay que hacérselo”.

5. esta es la primera parte de una regla que define qué elementos HTML deben ser estilizados con las propiedades y valores especificados en la regla.

6. Los selectores en CSS permiten identificar elementos en una página web para aplicarles estilos, como:

  • Selector de ID: Se utiliza para seleccionar un elemento único en el documento. Se define con el símbolo # seguido del identificador del elemento. Ejemplo: #miID.
  • Selector de clases: Se utiliza para seleccionar uno o más elementos que comparten una misma clase. Se define con el símbolo . seguido del nombre de la clase. Ejemplo: .miClase.
  • Selector universal: Selecciona todos los elementos del documento. Se representa con un asterisco (*). Ejemplo: * { margin: 0; }.
  • Selector de etiquetas: Selecciona todos los elementos de un tipo específico de etiqueta HTML. Ejemplo: p selecciona todos los párrafos.
  • Selector descendente: Selecciona elementos que son descendientes de un elemento específico. Se define separando los selectores por un espacio. Ejemplo: div p selecciona todos los párrafos dentro de un div.

7.

  1. El navegador carga el HTML (por ejemplo, lo recibe de la red).
  2. Convierte el HTML en un DOM (Modelo de objetos del documento). El DOM representa el documento en la memoria del ordenador. Lo explicaremos más detalladamente en la sección siguiente.
  3. Entonces, el navegador va a buscar la mayor parte de los recursos vinculados al documento HTML, como las imágenes y los videos incrustados...
    ¡y también el CSS vinculado! JavaScript aparece un poco más adelante en el proceso, pero no vamos a hablar de ello aún para evitar complicar las cosas.
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector. Por ejemplo, elemento, clase, ID, y así.
    Para cada tipo de selector que encuentre, calcula qué reglas deben aplicarse y a qué nodos en el DOM se les aplica el estilo según corresponda (este paso intermedio se llama árbol de renderización).
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
  6. En la pantalla se muestra el aspecto visual de la página (esta etapa se llama pintura).

8. las tres formas principales para vincular CSS a HTML: estilos en línea, estilos incrustados (o internos) y estilos externos.
